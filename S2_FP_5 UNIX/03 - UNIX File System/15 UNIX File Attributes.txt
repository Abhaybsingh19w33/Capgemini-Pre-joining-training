UNIX FILE ATTRIBUTES -> ATTRIBUTES OF A UNIX FILE
Every file has the following properties
-> Type
-> Ownership
-> Permission
-> Size
-> Timestamp

Different file types
-> Ordinary
-> Directory
-> Character
-> Symbolic link

Ownership of a file
-> Owner
-> Group of users

Different file permissions
-> Read
-> Write
-> Execute

Access categories
-> User
-> Group
-> Other

Timestamp on a file reveals
-> Creation time
-> Modified time
-> Access time
